# Data Connectors

Scripts and tools for extracting health data from portals, devices, and wearables.

- **[MitID / Sundhed.dk Integration](https://github.com/mikkelkrogsholm/ai-laegens-bord/tree/main/.claude/skills/sundhed-dk)**: A local script using Playwright to handle MitID automated browser logins to securely extract user health records. (Part of the AI-LÃ¦gens Bord project).

- **[apple-health-exporter](https://github.com/mganjoo/apple-health-exporter)**: Export Apple Health XML data to pandas DataFrames for analysis and integration with AI workflows.

- **[python-garminconnect](https://github.com/cyberjunky/python-garminconnect)**: Python 3 API wrapper for Garmin Connect enabling programmatic access to fitness device data and health metrics.

- **[HealthConnectExports](https://github.com/angeloanan/HealthConnectExports)**: Tool for exporting Android Health Connect data, providing interoperability between Android health apps and external systems.

- **[Vital2AI](https://github.com/agalbourdin/vital2ai)**: Converts Apple Health data into AI-ready CSV formats, streamlining the pipeline from device data to machine learning applications.

- **[GarminDB](https://github.com/tcgoetz/GarminDB)**: Aggregates Garmin and FitBit data into SQLite databases for persistent storage and structured querying of wearable health information.
