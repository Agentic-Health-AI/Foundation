# Research & Literature APIs

APIs and tools for searching and retrieving biomedical research literature.

- **[PubMed E-utilities](https://eutils.ncbi.nlm.nih.gov/entrez/eutils/)**: NCBI's comprehensive REST API providing programmatic access to PubMed and 35+ biomedical databases with advanced search and filtering capabilities.

- **[medRxiv Search Tool](https://github.com/mikkelkrogsholm/ai-laegens-bord/tree/main/.claude/skills/medrxiv-search)**: A script that queries the free CSHL API for medical preprints.

- **[PubMed Wrapper](https://github.com/mikkelkrogsholm/ai-laegens-bord/tree/main/.claude/skills/pubmed-database)**: Direct REST API access to PubMed with advanced search filters for programmatic agents.

- **[Europe PMC](https://europepmc.org/RestfulWebService)**: RESTful API providing access to over 33 million publications with text-mined annotations, including biological entities and disease mentions.

- **[Semantic Scholar](https://api.semanticscholar.org/)**: AI-powered research paper search and indexing across 214 million papers with automated citation and research impact analysis.

- **[OpenAlex](https://openalex.org/)**: Free and open catalog of scholarly works, venues, authors, and institutions, succeeding Microsoft Academic Graph with comprehensive coverage and no paywalls.

- **[bioRxiv/medRxiv API](https://api.biorxiv.org/)**: Direct access to preprint metadata for biology and medical research with usage statistics and content filtering options.

- **[paperscraper](https://github.com/jannisborn/paperscraper)**: Python library for multi-source preprint and publication scraping from PubMed, arXiv, bioRxiv, and medRxiv.
