# Skills & Connectors

This directory contains (or links to) open-source scripts, API wrappers, and Playwright automation tools that act as the "hands and eyes" of Agentic Health. These tools let agents fetch data, log into portals, and query research databases.

Want to add a new skill? Check the [CONTRIBUTING.md](../CONTRIBUTING.md).

## Featured Data Connectors

- **[MitID / Sundhed.dk Integration](#)**: A local script using Playwright to handle MitID automated browser logins to securely extract user health records.

## Research Connectors

- **[medRxiv Search Tool](#)**: A script that queries the free CSHL API for medical preprints.
- **[PubMed Wrapper](#)**: Direct REST API access to PubMed with advanced search filters for programmatic agents.
