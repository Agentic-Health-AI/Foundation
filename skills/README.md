# Skills & Connectors

This directory contains (or links to) open-source scripts, API wrappers, and Playwright automation tools that act as the "hands and eyes" of Agentic Health. These tools let agents fetch data, log into portals, and query research databases.

## Table of Contents

- [MCP Servers](mcp-servers.md) — Model Context Protocol servers for health data
- [Research APIs](research-apis.md) — Literature and research database APIs
- [Clinical Data APIs](clinical-data-apis.md) — FHIR, OMOP, and EHR integrations
- [Drug APIs](drug-apis.md) — Drug and pharmacology data sources
- [Genomics Databases](genomics-databases.md) — Genomics and variant databases
- [Terminology APIs](terminology-apis.md) — Medical terminology and ontology services
- [Data Connectors](data-connectors.md) — Health portal and data extraction tools
