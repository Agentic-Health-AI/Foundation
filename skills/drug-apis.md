# Drug & Pharmacology APIs

APIs and databases for drug information, interactions, and pharmacogenomics.

- **[OpenFDA](https://open.fda.gov/)**: Provides free access to FDA data on adverse events, drug labeling, NDC codes, and recalls. No authentication required.

- **[RxNorm](https://lhncbc.nlm.nih.gov/RxNav/APIs/index.html)**: NLM's drug name normalization and relationship service for standardizing drug terminology across different naming conventions. Free API, no authentication.

- **[DailyMed](https://dailymed.nlm.nih.gov/dailymed/app-support-web-services.cfm)**: Provides current and historical FDA-approved drug labeling and prescribing information via web services and data downloads.

- **[PharmGKB](https://www.pharmgkb.org/)**: Comprehensive pharmacogenomics knowledge base documenting relationships between genes, drugs, and phenotypes. Free academic access available.

- **[Open Targets](https://platform.opentargets.org/)**: Integrative platform for drug target identification and validation with a GraphQL API for querying drug-disease-target associations.

- **[DrugBank](https://go.drugbank.com/)**: Extensive drug database with information on drug mechanisms, interactions, and targets. Free academic data downloads available.
