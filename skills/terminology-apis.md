# Medical Terminology & Ontology APIs

APIs for standardized medical vocabularies, disease classifications, and concept mapping.

- **[UMLS Terminology Services](https://uts.nlm.nih.gov/)**: Comprehensive API for accessing 200+ medical vocabularies including SNOMED CT, ICD-10, MeSH, and RxNorm with free licensing.

- **[SNOMED CT](https://www.snomed.org/)**: International standard for clinical terminology with 350,000+ concepts. Free access in member countries including Denmark.

- **[ICD-11 API](https://icd.who.int/icdapi)**: WHO's official REST API for disease classification, mortality, and morbidity coding. Free registration required.

- **[MeSH (Medical Subject Headings)](https://id.nlm.nih.gov/mesh/)**: Biomedical vocabulary lookup via SPARQL endpoint and REST API with no authentication required.

- **[Orphanet API](https://api.orphacode.org/)**: Free REST API for rare disease nomenclature, classification, and cross-references to other medical standards.
