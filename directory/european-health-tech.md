# European Health Tech

Open-source health data infrastructure and standards from Europe.

- **[HealthData@EU](https://code.europa.eu/healthdataeu)**: EU open-source central platform implementing the European Health Data Space (EHDS) to enable secure health data exchange across member states.

- **[EHRbase](https://github.com/ehrbase/ehrbase)**: Leading open-source openEHR clinical data repository providing a standards-compliant platform for structured health record storage and retrieval.

- **[HAPI FHIR](https://github.com/hapifhir/hapi-fhir)**: Most widely adopted open-source FHIR server implementation, enabling interoperable health data exchange across systems and organizations.

- **[Blaze FHIR](https://github.com/samply/blaze)**: High-performance FHIR server developed by the German Medical Informatics Initiative, optimized for large-scale health data operations.

- **[Orthanc](https://github.com/jodogne/OrthancMirror)**: Belgian open-source DICOM server for medical imaging, providing lightweight and efficient storage and sharing of medical images.

- **[GNU Health](https://github.com/msuarez/gnuhealth)**: UN-recognized Digital Public Good hospital information system offering comprehensive health management capabilities for healthcare institutions worldwide.

- **[openIMIS](https://github.com/openimis)**: Open-source health insurance management system serving over 36.6 million beneficiaries, enabling efficient claims processing and beneficiary management.

- **[Samply](https://github.com/samply)**: German federated biobank query infrastructure enabling secure, distributed search and access to biological samples across multiple biobanks without centralizing sensitive data.
