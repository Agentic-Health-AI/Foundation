# Clinical Reasoning & Diagnostics

Tools for clinical decision support, diagnostic reasoning, and medical AI assistance.

- **[OpenMedicine](https://github.com/RamosFBC/openmedicine)**: DOI-traceable clinical reasoning MCP server enabling evidence-based diagnostic assistance with direct citation to medical literature.

- **[Medical Deep Research](https://github.com/Clinical-Copilot/Medical_Deep_Research)**: Agentic medical investigation system with MCP integration for comprehensive clinical research and diagnostic support.

- **[Multi-Agent Medical Assistant](https://github.com/souvikmajumder26/Multi-Agent-Medical-Assistant)**: Multi-agent diagnostic system leveraging graph workflows for coordinated clinical reasoning across specialized medical domains.

- **[AI-Agents-for-Medical-Diagnostics](https://github.com/ahmadvh/AI-Agents-for-Medical-Diagnostics)**: Multi-specialist consultation framework simulating collaborative diagnosis through autonomous medical agents.

- **[Medical-RAG-LLM](https://github.com/AquibPy/Medical-RAG-LLM)**: Complete medical retrieval-augmented generation pipeline combining domain-specific knowledge bases with LLM reasoning for clinical support.

- **[Doctor-Chatbot](https://github.com/LifestyleCorp/Doctor-Chatbot)**: Multimodal multi-agent clinical assistant integrating text, image, and audio inputs for comprehensive patient evaluation.

- **[Healthcare-aigent](https://github.com/pkuppens/healthcare-aigent)**: Modular multi-agent healthcare system built with CrewAI and LangFlow for flexible clinical workflow orchestration.

- **[Agentic Healthcare AI](https://github.com/amitpuri/agentic-healthcare-ai)**: FHIR and HIPAA-compliant agent framework designed for enterprise healthcare integration and regulatory compliance.
