# Literature Search & Review

Tools for searching, reviewing, and synthesizing biomedical literature.

- **[LatteReview](https://github.com/PouriaRouzrokh/LatteReview)**: Automated systematic review framework using multiple LLMs for comprehensive literature analysis and synthesis across biomedical research domains.

- **[PubMedSummarizer](https://github.com/malbiruk/PubMedSummarizer)**: Constructs knowledge graphs from PubMed literature to extract and visualize relationships between biomedical concepts and findings.

- **[paperai](https://github.com/neuml/paperai)**: Large-scale medical literature analysis tool enabling efficient searching, summarization, and pattern discovery across extensive biomedical datasets.

- **[paperscraper](https://github.com/jannisborn/paperscraper)**: Multi-source scraping tool that aggregates preprints and PubMed articles, providing unified access to diverse biomedical literature sources.

- **[LitLLM](https://github.com/LitLLM/LitLLM)**: Hybrid literature review system combining keyword-based and semantic search approaches for comprehensive biomedical literature discovery and analysis.

- **[Agent Literature Review](https://github.com/Arcadia-Science/agent-literature-review)**: Multi-agent collaborative framework for conducting systematic literature reviews with distributed task handling and synthesis capabilities.

- **[Sentinel](https://github.com/EbodShojaei/sentinel)**: PubMed search agent featuring intelligent MeSH query generation for enhanced biomedical literature retrieval and exploration.

- **[PubMed Article Search Agent](https://github.com/mewz1k/PubMed_Agent)**: LlamaIndex-based agent providing intelligent conversational interface for PubMed article discovery and analysis.
