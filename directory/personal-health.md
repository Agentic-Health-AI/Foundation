# Personal Health & Wearables

Tools for managing, analyzing, and acting on personal health and wearable data.

- **[AI-LÃ¦gens Bord (AI Doctor's Table)](https://github.com/mikkelkrogsholm/ai-laegens-bord)**: A collection of AI skills for personal health assistance in Denmark. Features a local-first workflow integrating with Claude Code that uses Playwright to securely log into Sundhed.dk via MitID, downloads the user's laboratory results, and cross-references them against recent studies in PubMed and medRxiv to flag optimal vs. normal ranges.

- **[OpenHealth](https://github.com/OpenHealthForAll/open-health)**: A local-first AI health assistant that helps users understand and manage their personal health data with privacy-preserving AI analysis.

- **[Fasten Health](https://github.com/fastenhealth/fasten-onprem)**: A self-hosted multi-provider health record aggregation platform that consolidates data from multiple healthcare systems into a unified personal health record.

- **[Open Wearables](https://github.com/the-momentum/open-wearables)**: A unified wearable API providing standardized access to data from multiple wearable device manufacturers with Model Context Protocol (MCP) support.

- **[Vital2AI](https://github.com/agalbourdin/vital2ai)**: Converts Apple Health data into AI-ready CSV format, enabling seamless integration with machine learning and analysis pipelines.

- **[GarminDB](https://github.com/tcgoetz/GarminDB)**: A Garmin data warehouse that stores activity, health, and sleep data with built-in Jupyter notebook support for exploratory analysis and visualization.

- **[BLOOD_TEST_ANALYSIS](https://github.com/cherry-1729-9090/BLOOD_TEST_ANALYSIS)**: A multi-agent AI system that performs comprehensive blood test analysis, interpreting results and flagging abnormalities with contextual health insights.

- **[Metriport](https://github.com/metriport/metriport)**: A universal healthcare data API that provides FHIR-compliant access to medical records from hundreds of healthcare providers and systems.

- **[wearablecompute](https://github.com/DigitalBiomarkerDiscoveryPipeline/wearablecompute)**: A biomarker feature engineering library that transforms raw wearable sensor data into clinically relevant health metrics and biomarkers.

- **[OpenMed](https://github.com/ianrowan/OpenMed)**: An agentic personal medical assistant that combines AI with health data to provide personalized health insights and recommendations.
